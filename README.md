### Installation of packages
1. Python 3.10.5
2. Install virtualenv
```pip install virtualenv```
3. build env
```virtualenv myenv```
4. open env
Windows ```.\venv\Scripts\myenv```
MacOS ```$ source ./venv/bin/myenv```
